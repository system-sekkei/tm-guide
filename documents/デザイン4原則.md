# デザイン4原則

1. 近接
    - 目的
      - 関係性（遠近）の表明
    - 手段
      - 余白(margin、padding)
    - 実装
      - Layout
        - [Gutters](https://getbootstrap.com/docs/5.3/layout/gutters/)
      - Utilities
        - [Spacing](https://getbootstrap.com/docs/5.3/utilities/spacing/)
      - Components
        - [Card](https://getbootstrap.com/docs/5.3/components/card/)
        - [Button group](https://getbootstrap.com/docs/5.3/components/button-group/#nesting)
2. 整列
    - 目的
      - 関係性（グループ）の表明
    - 手段
      - 左揃え、中央揃え、右揃え
    - 実装
      - Layout
        - [Grid](https://getbootstrap.com/docs/5.3/layout/grid/)
        - [Columns](https://getbootstrap.com/docs/5.3/layout/columns/)
        - [Gutters](https://getbootstrap.com/docs/5.3/layout/gutters/)
      - Content
        - [Tables](https://getbootstrap.com/docs/5.3/content/tables/)
      - Utilities
        - [Spacing](https://getbootstrap.com/docs/5.3/utilities/spacing/)
      - Utilities
        - [VerticalAlign](https://getbootstrap.com/docs/5.3/utilities/vertical-align/)
3. 強弱
    - 目的
      - 強調の表明
    - 手段
      - フォントサイズ、種類、写真の大きさの変化
    - 実装
      - Layout
        - [Typography](https://getbootstrap.com/docs/5.3/content/typography/)
      - Utilities
        - [Borders](https://getbootstrap.com/docs/5.3/utilities/borders/)
        - [Colors](https://getbootstrap.com/docs/5.3/utilities/colors/)
        - [Text](https://getbootstrap.com/docs/5.3/utilities/text/#font-weight-and-italics)
      - Extend
        - [Icons](https://getbootstrap.com/docs/5.3/extend/icons/)
4. 反復
    - 目的
      - 一貫性、統一性の表明
      - ページ内、ページ間の一貫性
    - 手段
      - 同じデザインルールの繰り返し
    - 実装
      - Forms
        - [Form controls](https://getbootstrap.com/docs/5.3/forms/form-control/)
      - Components
        - [Navbar](https://getbootstrap.com/docs/5.3/components/navbar/)
        - [List group](https://getbootstrap.com/docs/5.1/components/list-group/)
